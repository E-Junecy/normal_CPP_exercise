玫瑰花圃

题目描述

有一块n×n（n≥5，且 n 是奇数）的红玫瑰花圃，由 n×n 个小正方形花圃组成，现要求在花圃中最中间的一行、最中间的一列以及 

4 个顶点处种植粉色玫瑰，请问粉玫瑰占地面积占整个玫瑰花圃总面积的百分比是多少？



如 n=5，花圃形状如下，粉玫瑰占地百分比 =13/25=52.0%

粉红粉红粉

红红粉红红

粉粉粉粉粉

红红粉红红

粉红粉红粉



如 n=7，花圃形状为如下，粉玫瑰占地百分比 =17/49=34.7%

粉红红粉红红粉

红红红粉红红红

红红红粉红红红

粉粉粉粉粉粉粉

红红红粉红红红

红红红粉红红红

粉红红粉红红粉



输入

一个整数 n（5≤n≤99，且 n是奇数）



输出

粉玫瑰占总玫瑰园的百分比，结果保留 1 位小数，显示为百分比的形式。



样例

输入

5

输出

52.0%
